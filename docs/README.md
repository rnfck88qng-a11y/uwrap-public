# Docs

This directory separates public doctrine and research papers from core implementation material.

## Layout

- `papers/` — public-facing working papers, appendices, contributor briefs, and review material
- `doctrine/` — short doctrine pages that define stable public concepts

## Intent

The material here is additive and descriptive. It does not replace core specs, schemas, or implementation documents.

The paper lane is intentionally separated from implementation because the doctrine is more stable than the current stack shape.
