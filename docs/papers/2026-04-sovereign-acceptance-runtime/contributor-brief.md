# Contributor Brief
## The Sovereign Acceptance Runtime

This is a doctrine paper backed by bounded proofs.

It is not a finished platform announcement.

## What this work is about

The paper argues that AI governance needs a bounded execution layer, not just model moderation, dashboards, or policy language.

That layer is called the **Sovereign Acceptance Runtime**.

Its core principles are:
- AI may propose
- actions must be receipted
- contradictions must remain visible
- projection is not truth
- only humans accept canonical state

## Stable enough to build on

The following are stable enough to discuss and build around:

- Three Sacred Promises
