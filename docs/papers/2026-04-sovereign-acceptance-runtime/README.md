## Positioning

This is a doctrine paper, not a finished-system announcement.

It publishes:
- Sovereign Acceptance as a runtime discipline
- Tri-State Truth as an honesty protocol
- receipt-driven governance as a proof grammar
- STRONG/WEAK guard separation as a bounded execution pattern

It does not claim:
- a finished platform
- reconciled subsystem boundaries
- generalized CANON throughput
- universal artifact gating
- universal receipt unification
- complete sandboxing
